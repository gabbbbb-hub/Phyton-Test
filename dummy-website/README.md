# Dummy Website

A small static site for testing and demos.

How to run

1. Make sure you have Python 3 installed.
2. From this folder run:

```powershell
python server.py
```

3. Open http://127.0.0.1:8000 in your browser.

Files
- `index.html` – main page
- `styles.css` – styles
- `script.js` – small JS to populate cards
- `server.py` – tiny static server
- `assets/logo.svg` – demo logo

Notes
- The server prints the folder it's serving and stops with Ctrl+C.
